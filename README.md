## 🔧 How to Use

1. **Clone the repository:**

   ```bash
   git clone https://github.com/UllasDas004/DSA-WITH-CPP-Learning-
   cd DSA-WITH-CPP-Learning-
   ```

2. **Navigate to any topic folder:**

   ```bash
   cd 14_Binary_Search/
   ```

3. **Compile & Run:**

   ```bash
   g++ binary_search.cpp -o bs
   ./bs
   ```
